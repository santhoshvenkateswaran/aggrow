How do we predict crops?

This application is a user-friendly recommendation system for predicting the perfect crop according to

  • Rainfall
  • Temperature
  • Ph

in your desired land.

This application uses a supervised Machine Learning Ensemble Model comprising of 3 algorithms:

  • Support Vector Classifier
  • Decision Tree
  • Gradient Boosting

This is an end-to-end app with an aesthetically simple UI in which the user enters the values for the conditions (Rainfall, temperature & pH) and the three top-most suitable crops are displayed.
